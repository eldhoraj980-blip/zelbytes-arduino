# Methodology

1. Connect button, relay and LED to Arduino.
2. Connect solenoid valve through relay contacts.
3. Configure all pins in config.h.
4. Implement button debounce logic.
5. Read button state continuously.
6. Open valve while button is pressed.
7. Turn ON status LED when valve is open.
8. Log irrigation events with timestamp.
9. Verify relay operation.
10. Test complete irrigation cycle.
