# Task 3 - Manual Irrigation Toggle (Phase 1 Capstone)

## Objective
Integrate button, relay, status LED and serial monitoring into a manual irrigation controller.

## Requirements
- Single button controls irrigation
- Relay controls solenoid valve
- Status LED indicates valve state
- Serial Monitor logs irrigation events
- Timestamp generated using millis()
- Git release tag: v0.1-manual

## Hardware Used
- Arduino UNO
- Push Button
- Relay Module
- Solenoid Valve
- LED
- Power Supply

## Features
- Debounced button input
- Manual irrigation control
- Relay switching
- Valve status indication
- Event timestamp logging
