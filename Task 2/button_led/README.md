# Day 3 – Debounced Button and LED

## Objective
Toggle an LED using a push button with software debouncing.

## Pin Assignment

LED  -> Pin 13
Button -> Pin 2

## Features

- INPUT_PULLUP mode
- Non-blocking debounce
- LED toggle
- Serial logging

## Expected Output

Each button press toggles LED state.

Serial Monitor:

Toggle
Toggle
Toggle
