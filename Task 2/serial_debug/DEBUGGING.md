# Serial Debugging Workflow

## Connection

1. Connect Arduino through USB.
2. Upload sketch.
3. Open Serial Monitor.
4. Select 9600 baud.

## Commands

? = Help
l = LED ON
o = LED OFF

## Debug Messages

System prints uptime every 2 seconds.

## Troubleshooting

### No Output

Check:
- COM Port
- Baud Rate
- USB Cable

### Unknown Command

Verify command character entered.
